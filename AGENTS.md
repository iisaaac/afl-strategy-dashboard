# Project Standards For Future Codex Prompts

- Keep code modular and place reusable logic inside the `afl_strategy_dashboard` package.
- Add or update tests for new feature logic.
- Avoid betting, gambling, tipping and fantasy-football framing.
- Write insights in an executive style suitable for Strategy, Data & Analytics, Business Operations, Commercial, Stadium Operations, Digital/Media/Marketing and Technology/AI audiences.
- Use public data only.
- Do not use private or protected AFL, club or Champion Data sources.
- Maintain clear methodology documentation whenever metrics or scoring logic changes.
- Keep the Streamlit app professional, clean and recruiter-ready.
- Prefer simple, reliable pandas transformations over unnecessary abstraction.
